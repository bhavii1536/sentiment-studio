# sentiment-studio
Python project

Sentiment Studio is a multilingual sentiment analysis dashboard that analyzes
user-generated content in **English, Tamil, and Hindi** using transformer-based models.

## Features
- CSV-based sentiment analysis
- Automatic language detection
- Transformer models (RoBERTa, XLM-RoBERTa)
- Interactive visualizations
- Downloadable sentiment results

## Tech Stack
- Python
- Streamlit
- Hugging Face Transformers
- Pandas, Matplotlib

## Project Status
Phase 1 completed.
Upcoming features include YouTube and Twitter analysis, aspect-based sentiment analysis,
and creator analytics.
